# MealSync - TypeScript Edition

A React Native Expo app built with TypeScript for managing meal menus with child-specific dietary suggestions.

## Features

- **TypeScript**: Full type safety throughout the application
- **Meal Menu Screen**: Browse active meals with child-specific suggestions
- **Admin Screen**: Toggle meal active/inactive status with instant updates
- **Smart Suggestions**: Top 3 meals suggested based on child's allergens and dietary preferences
- **Real-time Updates**: Changes in admin panel reflect immediately in meal menu

## TypeScript Benefits Implemented

- ✅ Strict type checking with `strict: true` in tsconfig
- ✅ Proper interface definitions for all data models
- ✅ Type-safe context providers and custom hooks
- ✅ Component props fully typed
- ✅ Navigation type safety with RootStackParamList
- ✅ Module declarations for third-party libraries
